## Getting Started

1. Clone the repository
2. Run `hatch run setup` for automatic environment setup
3. Make your changes
4. Run `hatch run ci` to verify all checks pass
5. Submit a pull request

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for detailed instructions.
