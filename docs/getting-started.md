# Getting Started

## Installation

Install Structum using pip:

```bash
pip install structum
```

## Basic Usage

### Visualize Directory Tree

```bash
structum tree .
```

### Archive Code

```bash
structum archive . --output code_context.md
```

### Clean Artifacts

```bash
structum clean .
```
